# Roadmap Autonomous Research

## Foundation
- Lock strategy defaults behind schema and contract tests.
- Freeze universes, configs, manifests, and experiment ledgers.
- Keep provider abstraction narrow so higher-quality data can replace AKShare later.

## Next
- Expand tradability with richer exchange flags.
- Add higher-fidelity walk-forward and parameter robustness sampling.
- Separate research datasets by snapshot instead of reusing the same rolling database.

## Later
- Weekly / minute-level interfaces only after daily research remains auditable.
- Multi-strategy orchestration only after each strategy has a defensible promotion path.
