# 2026Q1_mom Research Spec

## Objective

Build a reproducible A-share momentum strategy workflow with project-scoped outputs:
- universe construction
- bars update to SQLite
- momentum rank generation
- TopN rebalancing backtest
- report and manifest outputs

## Universe Definition

- Universe size target: `3063` symbols
- Market scope: Shanghai/Shenzhen mainboard only (codes starting with `0` or `6`)
- Exclusions:
  - ST / *ST / delisting-tag stocks based on stock name snapshot
  - ChiNext (`300` / `301`)
  - STAR (`688` / `689`)
  - BSE-related prefixes (`4` / `8`)
- Frozen source of truth:
  - `data/projects/2026Q1_mom/meta/universe_codes.txt`

## Data & Storage

- Data source: AKShare daily A-share bars, adjusted `qfq`
- Frequency: `1d`
- Storage: SQLite `data/market.db`, table `bars`
- Project-scoped metadata:
  - `data/projects/2026Q1_mom/meta/`

## Signal Definition

- Momentum factor: `mom20 = close[t] / close[t-20] - 1`
- Rebalance frequency: every `5` trading days
- Selection: rank by `mom20`, choose Top-K (`K=5`)
- Candidate count tracking:
  - `data/projects/2026Q1_mom/meta/rank_candidate_count.csv`

## Backtest Rules

- Portfolio type: equal weight over selected TopN names
- Default TopN sweep: `Top1..Top5`
- Initial cash: `1,000,000`
- Cost assumptions:
  - commission: `0.03%` both sides
  - stamp duty: `0.10%` sell side
  - slippage: `0.05%`

## Required Outputs

- Rank file: `data/projects/2026Q1_mom/signals/rank_top5.parquet`
- Metrics: `artifacts/projects/2026Q1_mom/summary_metrics.csv`
- Curve plot: `artifacts/projects/2026Q1_mom/topn_1_5.png`
- Report: `artifacts/projects/2026Q1_mom/report.md`
- Manifest: `data/projects/2026Q1_mom/meta/run_manifest.json`

## Known Limitations

- ST filtering uses a current-name snapshot, not historical daily ST status reconstruction.
- Tradability constraints are simplified by default; richer filters are controlled by config and can be expanded.
- Data quality depends on AKShare coverage and local SQLite completeness.

## Reproducibility Contract

Every run must record:
- config parameters (`configs/projects/2026Q1_mom.json`)
- frozen universe size and path
- rank dates and candidate pool stats
- db coverage stats
- git commit (when available)

All run metadata is merged into:
- `data/projects/2026Q1_mom/meta/run_manifest.json`
