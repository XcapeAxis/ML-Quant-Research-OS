# Plain Language Glossary

- `auto-refreeze` / `refreeze`: automatically shrink and re-fix the current stock universe so it only keeps names with usable validated data.
- `research readiness`: whether the data is good enough to start serious research and strategy comparison.
- `coverage-gap ledger`: a data-gap list that explains which symbols are missing data and why.
- `validated universe`: the stock range that passed the current data checks.
- `pilot`: a partial recovery or trial state that is not yet strong enough for formal research conclusions.
- `experiment graph`: a structured experiment record that shows what data was used, what changed, and what result came out.
- `strategy failure report`: a report that explains why the strategy failed instead of only saying that it failed.
- `shadow execution`: a simulated execution interface, not real order placement.
