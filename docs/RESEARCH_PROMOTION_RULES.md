# Research Promotion Rules

Candidates are promotable only when all checks below pass:

1. Leakage audit passes.
2. Walk-forward validation survives enough windows.
3. Cost sensitivity retains enough of the base return.
4. Max drawdown does not exceed 30%.
5. The research hypothesis is explicit and economically interpretable.
6. Parameter perturbation coverage exists; single-window / single-parameter luck is not enough.

Promotion reports must explain why a candidate passed or failed. Failed promotions still belong in the experiment ledger.
